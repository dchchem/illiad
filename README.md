# ILLIAD - Precise Layered Molecular Packing Metrics Evaluation

ILLIAD is a Python script dedicated to a precise calculation of various descriptors and metrics for crystal structures with a layered packing. The script can be run from command line both with and without initial parsing.
The program reads the SHELXL ```.res``` file with the asymmetric unit cell **already grown** (if the molecule lies on the special position, e.g. mirror plane, inversion center, rotoational axis, etc.) and prints out some crucial intra- and interlayered packing metrics.

Example of command line execution:

- ```-h``` - print the usage instructions
- ```-hkl``` - Miller (hkl) indices for a layer (need to be calculated by user a priori, e.g. with Mercury, XP or Olex2)
- ```-m``` - number of molecules on layer per unit cell
- ```-r``` - van der Waals sphere resolution (number of vertices per unit area)
- ```-q``` - quiet mode with no verbose until final results are printed
- ```-sect``` - plot the layer cross-section of a molecular van der Waals surface
- ```-surf``` - plot the molecular van der Waals surface
- ```-scld``` - plot the point cloud for molecular van der Waals surface
- ```-p``` - project name
```
python illiad.py [-h] [-hkl h k l] [-m MOL] [-r RESOLUTION] [-q] [--sect SECT] [--surf SURF] [--scld SCLD] [-p PROJECT]
```

## Example
We will use the RES file from the CCDC (refcode FITXIP, no. 1157255) of 3,6-diamino-1,2,4,5-tetrazine. The whole molecule lies on a special position generated by a mirror plane, so the asymmetric unit consists of exactly half of the molecule. For proper use of ILLIAD, we must generate the second half of the molecule and save it in the RES file as well. To do it, we can use SHELXP:
```
XP>> REAP FITXIP.res
XP>> FMOL

 N1  - C1
 N2  - C1
 C1  - N1   N2   N3
 N3  - C1   H1   H2
 H1  - N3
 H2  - N3

XP>> GROW
XP>> FILE FITXIP.res

 Enter name of file from which instructions (including HKLF but not atoms) should be
 copied [FITXIP.res]:
** warning: N1A  is a symmetry equivalent of N1
** warning: N2A  is a symmetry equivalent of N2
** warning: C1A  is a symmetry equivalent of C1
** warning: N3A  is a symmetry equivalent of N3
** warning: H1A  is a symmetry equivalent of H1
** warning: H2A  is a symmetry equivalent of H2
```
The warning signs for symmetry equivalent generation shows us that the second half of the molecule has been generated and written to the RES file. After that, we can use ILLIAD:
```
python illiad.py -p FITXIP -hkl 1 0 0 -m 2 --sect True
```
We have parsed the RES file along with the layer Miller indices (100) and a number of molecules in a unit cell at that layer (a total of 2). Since we called the ```--sect``` keyword, we will see the section of van der Waals surface by that layer:
<img width="320" alt="Figure_1" src="https://github.com/user-attachments/assets/61449113-3240-4a24-9cf1-45e703b86751" />

The final lines contain the calculated metrics:
```
---------- FINAL RESULTS ----------
Metric               GZZ  With correction
----------------  ------  -----------------
V(vdW), Ang^3     85.224  ---
S(m,s), Ang^2     25.066  36.632
C(m,s), Ang       17.748  26.181
d(a,s), Ang^(-2)   0.479  0.328
ILMPC              0.679  0.992
PC                 2.95   ---
d(IL), Ang         6.257  ---
```
The majority of these metrics ($S_{m,s}$, $C_{m,s}$, $d_{a,s}$, ${\rm ILMPC}$) are calculated based on the 2024 paper by Guo, Zhu and Zhang (*Cryst. Growth. Des.* 2024, **24**, 9849-9856) - they are listed below the GZZ column. The rest of them are the van der Waals surface inner volume ($V_{\rm vdW}$), (standard) packing coefficient (PC) and interlayer distance ($d_{\rm IL}$). The second column shows the values based on the acylindrical approximaton. These metrics can be useful when the whole molecule isn't planar and one (or multiple) functional groups "stand out" of the layer.
